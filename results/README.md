# Results layout

`results/figures/` contains versioned publication PDFs. Simulation caches, MATLAB inputs, EMNIST data, and per-figure intermediates are excluded from Git and should be downloaded from `ZENODO_RECORD_URL` or regenerated.

Place downloaded HDF5 caches in `sim_files/`, MATLAB inputs in `matlab/`, and leave per-figure outputs in their matching `Fig_*` directory.
